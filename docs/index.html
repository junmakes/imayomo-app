あとで読むを、いま読もうへ。

「あとで読む」サービスを使ってはいるが、あとで読むことがほとんどないあなたへ。 読む記事に悩まない、読むことを少しだけ継続できるアプリ。 「あとで読む」サービスのPocketと連携して、Pocketに保存している記事を読むサポートをします。

主な機能

読む記事に悩まない

記事を読むには「読む」ボタンを押すだけ。たくさんの選択肢を見せて「決断疲れ」を起こさせるようなことはしません。

日々の読了状況を可視化

数値やグラフを使ってあなたの日々の読了状況を可視化することで、読むことを継続できていると実感させます。

目標への達成感を得られる

1週間または日単位で読了数の目標値を設定することで、目標達成に向けてのモチベーションを上げることができます。

アプリを起動して「読む」ボタンを押すだけ

どの記事から読むか迷わせません。 「あとで読む」リストに並んだたくさんの読む候補から、どれを読むかに悩んだり、さらに後回しにしたりしていませんか？ imayomoでは、あなたはただアプリを起動して「読む」ボタンを押し、表示されたサイトを読むだけです。

今どのくらい継続できているか確認できる

「読むことが継続できている」と実感させます。 日々どれくらい「あとで読む」されて、実際に読んでいるか把握していますか？ imayomoでは、読了の状況が直感的に理解できるので、日々の目標数達成など改善の具体的なアクションに繋げられます。

目標の達成度がすぐわかる

読了数に目標を設定することで達成へ向けてモチベーションが上がります。 達成可能な具体的な目標値が見えていると、達成したくなりませんか？ 1週間または日単位で読了数の目標値が設定でき、目標達成に必要な残りの読了数をトップ画面とバッジで確認できます。
